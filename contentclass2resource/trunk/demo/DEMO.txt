Демонстрационный план для продукта contentclass2resource/demo:

    Используемое ПО:
    
        contentclass2resource
        
    Предварительная подготовка:

        нет

    Демонстрационный план:
    
        1. Войдите в созданный экземпляр "Test content", имеющий имя
           test_content. Перейдите на вкладку View. Значения отображаемых
           параметров должны выглядеть так:

            Class name: Тестовый класс
            Class icon: none.png

           Они показывают значения ресурсов (параметров), общие для всех
           экземпляров данного класса. На этой вкладке значения параметров
           отображаются с помощью вида @@resource контент-класса.
        
        2. Перейдите на вкладку Demo. На ней должны отображаться поля выбора,
           "Russian name" и "Class icon", заполненные значениями параметров из
           соответствующих словарей ресурсов.
        
        3. Выйдите в корень дерева ZMI, путём щелчка по ссылке "основная",
           находящейся в верхней правой части страницы. Войдите в экземпляр
           "Resource container", имеющий имя cont_res.
        
        4. Войдите в объект contentclass2resource.demo.mycontentclass.MyContentClass
           и измените его параметры так, чтобы они получили следующие значения:

            Class name: Изменённый тестовый класс
            Class icon: testing.png

           Нажмите кнопку "Изменить".
        
        5. Выйдите в корень дерева ZMI, путём щелчка по ссылке "основная",
           находящейся в верхней правой части страницы. Войдите в экземпляр
           "Test content", имеющий имя test_content. Перейдите на вкладку View.
           Значения отображаемых параметров должны выглядеть так:

            Class name: Изменённый тестовый класс
            Class icon: testing.png

           Таким образом, при изменении значений ресурсов в Resource container'е
           изменяются значения ресурсов экземпляров соответствующего класса.
