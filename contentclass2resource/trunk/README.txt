Как использовать продукт contentclass2resource:

    Продукт предназначен для получения ресурсов класса по его наименованию.
    При добавлении экземпляра продукта (Resource container) указывается значение
    префикса. После добавления Resource container для каждого из ресурсов,
    перечисленных в IContentClassResource, будет зарегистрирован словарь с
    именем "префикс имя_ресурса", содержащий пары
    "имя контент-класса" - "значение ресурса для этого контент-класса".
    
    Например, если префикс равен "rescon_", а имя ресурса - "icon", то имя, под
    которым будет зарегистрирован соответствующий словарь, будет "rescon_icon".
    
    После добавления экземпляра Resource container в дерево объектов, в этот
    контейнер можно добавлять экземпляры Resource content, каждый из которых
    соответствует определённому контент-классу и обладает полями,
    соответствующими набору ресурсов. Сразу после добавления экземпляра
    Resource content значения соответствий наименования контент-класса и
    значений ресурсов попадают в зарегистрированные словари.

    В дальнейшем эти словари могут использоваться для выбора значений ресурсов,
    соответствующих определённому контент-классу. Пример реализации можно
    посмотреть в файле interfaces.py, класс IMyContentClass.
