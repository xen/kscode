Установка и настройка продукта interfaceswitcher:

    Используемое ПО:

        нет

    Список плагинов:

        demo

    Предварительная подготовка:

        нет

    Установка:

    1. Папку с продуктом скопировать в $INSTANCE/lib/python

    2. Из папки etc продукта скопировать файл interfaceswitcher-configure.zcml в
       $INSTANCE/etc/package-includes

    3. Перезапустить Zope

    4. Если предполагается демонстрация продукта, то следует установить
       плагин demo согласно содержащейся в нем инструкции.
