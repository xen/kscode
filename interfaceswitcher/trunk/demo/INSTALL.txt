Установка и настройка плагина demo продукта interfaceswitcher:

    Используемое ПО:

        interfaceswitcher

    Список плагинов:

        нет

    Предварительная подготовка:

        Для успешной установки demo должны
        быть предварительно установлены следующие продукты:

            interfaceswitcher

    Установка:

        1. Скопировать из папки плагина файл etc/interfaceswitcher-demo-configure.zcml
           в ZOPEINSTANCE/etc/package-includes.

        2. Перезапустить Zope.

        3. В корне сайта создать объект Catalog.

        4. В корне сайта создать объект Folder.

Конец документа
