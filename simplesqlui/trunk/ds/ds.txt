  Аннотация: 
  
    Компонент ds (источник данных), позволяет получать данные по какому-либо
    объекту из базы данных и модифицировать их. В простейшем случае, объект 
    класса  ds соответствует одной таблице базы данных.

  Идея реализации:

    Компонент представляет собой контейнер, который содержит приблизительно
    следующий список ZSqlMethod-ов (список может быть модифицирован в 
    процессе разработки):
    
        select -- осуществляет отбор одного объекта, обычно, данные
            объекта выбираются из базы полностью;
            
        select_all -- осуществляет отбор всех объектов базы, как правило,
            для каждого объекта отбирается только часть данных, на список
            объектов могут быть наложены дополнительные ограничения;
            
        update -- обновляет содержимое объекта, список аргументов
            может варьироваться;

        delete -- удаляет один или несколько объектов, список которых
            передан как аргумент, если select_all содержит ограничения
            на список отдаваемых объектов, аналогичные ограничения
            должны реализовываться и здесь.

    Каждый из DS регистрируется с именем, используемом в последствии для 
    идентификации связи в объектах класса record.

            
  Интерфейсы:
    
    IDS -- содержит методы select, select_all, update и
        delete, которые являются обертками для вызова соответствующих
        SqlMethod'ов;
        
    IContainer -- вид содержимого, позволяет добавлять SQLMethod'ы';
    
    IDSProp -- свойтсва: db (коннектор к базе данных).
