  Аннотация: 
  
    Компонент factory позволяет создавать утилиты, регистрируемые
    в системе, и используемые для динамического создания объектов, 
    производных от record, и включения в них ссылок на ds и 
    уникальных идентификаторов записей.

  Идея реализации:

    Factory предоставляет специальный метод, generate(), который
    получает на вход запись data, полученную из метода select продукта
    ds, и объект, который будет родителем получаемого объекта.
    
    Далее, в реестре утилит находится фабрика для класса, производного
    от record, название которой берется из атрибута factory объекта Factory.
    Оттуда же берется ds для объектов этого типа.
    
    Далее: RRecord(__parent__,ds,data,id)
    
    id - собственно тоже список идентификаторов. Вот, примерно так.
    
  Интерфейсы:
  
    ISqlFactory -- собственно метод generate(),
    
    ISqlFactoryProp -- свойства: ds, список id, factory, к интерфейсу
        есть вид для редактирования свойств.
