Установка и настройка продукта reference:

    Используемое ПО:
    
        нет
        
    Список плагинов:
    
        demo
        
    Предварительная подготовка:
        
        До начала инсталляции, должен существовать экезмпляр продукта uniqueid,
        зарегистрированный под пустым ("") именем.
             
    Установка:
    
        1. До начала инсталляции, должен существовать экезмпляр продукта
           uniqid, зарегистрированный под пустым ("") именем
        
        2. Папку с продуктом скопировать в $INSTANCE/lib/python;
        
        3. Из папки etc продукта скопировать файл reference-configure.zcml в
           $INSTANCE/etc/package-includes;
        
        4. Перезапустить zope
